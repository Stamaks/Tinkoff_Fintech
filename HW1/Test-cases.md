__Тест-кейс 1.__ Работоспособность формы.

__Шаги.__
1. Ввести любую фамилию, имя и отчество в поле "Фамилия, имя, отчество*", например, Иванов Иван Иванович.
2. Ввести любую электронную почту, например a@ya.ru
3. Ввести любой номер телефона, например +79999999999
4. Ввести любой город, например, Москва.
5. Выбрать любую вакансию, например, Специалист отдела взыскания.
6. Поставить галочку в чек-боксе.
7. Нажать кнопку отправить.

__Ожидаемый результат.__
Введенные данные отправились на сервер.

__Тест-кейс .__ Список с предложенными фамилиями прокручивается.

__Шаги.__ 
1. Ввести в поле ФИО любую букву русского алфавита, кроме й, ъ, ь, ы.
2. Дождаться появления выпадающего списка с предложенными фамилиями.
3. Навести указатель мыши на список.
4. Крутить колесико мышки.

__Ожидаемый результат.__
В зависимосит от движений колесика мышки список будет прокручиваться.

---

__Тест-кейс .__ Список с предложенными фамилиями прокручивается стрелками.

__Шаги.__ 
1. Ввести в поле ФИО любую букву русского алфавита, кроме й, ъ, ь, ы.
2. Дождаться появления выпадающего списка с предложенными фамилиями.
3. Начать нажимать клавиши вниз или вверх на клавиатуре.

__Ожидаемый результат.__
В зависимосит от нажатий клавиш список будет прокручиваться.

---

__Тест-кейс .__ Из списка с предложенными фамилиями можно выбрать.

__Шаги.__ 
1. Ввести в поле ФИО любую букву русского алфавита, кроме й, ъ, ь, ы.
2. Дождаться появления выпадающего списка с предложенными фамилиями.
3. Навести на выпавший список курсов мышки.
4. Выбрать любую фамилию (если надо, воспользоваться колесиком для прокручивания списка).
5. Кликнуть по выбранной фамилии.

__Ожидаемый результат.__
Выбранная фамилия появится в поле ФИО.
