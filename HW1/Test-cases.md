__Тест-кейс .__ Цифры в поле "ФИО".

__Шаги.__
1. В поле "ФИО" ввести строку из кирилических или латинских заглавных и маленьких букв, содержащую цифры, например "Иван0123456789 Иванович".

__Ожидаемый результат.__
В поле будут отсутствовать цифры (получится, например, "Иван Иванович").

Тест-кейс со специальными символами в поле ФИО намеренно не был описан, потому что в данной форме на сайте не существует проверки на специальные символы.

__Тест-кейс .__ Неправильный код города в поле "Номер телефона".

__Шаги.__
1. Ввести любой номер телефона, начинающийся со следующих цифр: 0, 1, 2, 7, например, 0000000000.
2. Кликнуть в любое место вне этого поля.

__Ожидаемый результат.__
У поля появится красная рамка, крестик справа и подпись снизу "Код города/оператора должен начинаться с цифры 3, 4, 5, 6, 8, 9".

__Тест-кейс .__ Попытка ввести слишком много цифр в поле "Номер телефона".

__Шаги.__
1. В поле "Номер телефона" ввести любую строку, содержащую больше 10-11 цифр, например "91234567890987654321".

__Ожидаемый результат.__
В поле будут введены только те первые несколько цифр, которые не выходят за формат длины номера телефона (для приведенного примера - 9123456789)

---

__Тест-кейс .__ Попытка ввести номер телефона, начинающийся на +7 в поле "Номер телефона".

__Шаги.__
1. В поле "Номер телефона" ввести любую строку, начинающуюся на "+7" и состоящую только из цифр.

__Ожидаемый результат.__
В поле будут введены только те цифры, которые находятся после "+7" и лишь столько, столько подходит по формату номера телефона.
